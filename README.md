# Ounin - Premium Steam Ovens Website

A complete, responsive e-commerce website for Ounin — a premium steam oven brand. Built with pure HTML, CSS, and JavaScript with no external CDN dependencies.

## 🌐 Live Demo

[Visit Ounin](https://yourusername.github.io/ounin-clone)

## ✨ Features

- **Responsive Design** — Mobile-first approach, works on all devices
- **Full E-commerce Experience**
  - Product catalog with filtering
  - Shopping cart with localStorage persistence
  - Quick view product modals
  - Wishlist functionality
- **Interactive UI**
  - Smooth scroll navigation
  - Fade-in animations on scroll
  - Toast notifications
  - Modal system
- **Recipe Section** — Chef-curated recipes with detailed instructions
- **Search Modal** — Global search across products and recipes
- **Contact Form** — Functional form with validation
- **Newsletter Subscription** — Email signup with confirmation
- **Modern Design** — Elegant serif headings, warm color palette, premium feel

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Flexbox, Grid, animations
- **Vanilla JavaScript** — No frameworks, no dependencies
- **LocalStorage** — Cart and wishlist persistence

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ounin-clone.git
   cd ouning-clone
   ```

2. **Open in browser**
   - Simply open `index.html` in your browser
   - Or use a local server:
     ```bash
     npx serve
     # or
     python -m http.server 8000
     ```

3. **Customize**
   - Replace placeholder images in `assets/images/`
   - Update brand name, colors, and content in HTML files
   - Modify product data in `main.js`
